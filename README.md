# 🎮 AI Tic Tac Toe

This is a Python-based Tic Tac Toe game with an AI opponent using the Minimax algorithm.

## 🚀 Features
- GUI using Tkinter
- AI opponent (unbeatable)
- Score tracking
- Win highlight system

## ▶️ How to Run
Run the file:
python main.py

## 🧠 Concepts Used
- Minimax Algorithm
- GUI Programming (Tkinter)
